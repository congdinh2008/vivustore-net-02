# Run commands

## Run application with dotnet cli

1. Restore the solution

    ```bash
    dotnet restore
    ```

2. Build the solution

    ```bash
    dotnet build
    ```

3. Run ViVuLMS.WebAPI

    ```bash
    dotnet run --project ViVuLMS.WebAPI
    ```

4. Dotnet Watch
    
    ```bash
    dotnet watch --project ViVuLMS.WebAPI
    ```
